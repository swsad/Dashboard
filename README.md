# 团队开发记录



## 文档

**命名规范**：用`-`连接编号和文档名，即`XX-XX-doc.md`，并且所有字母小写，例如`2-team-profile.md`

**文档链接**：在`index.md`中添加链接，网址为`https://github.com/swsad/Dashboard`加上去掉`.md`的文档名，例如`https://github.com/swsad/Dashboard/1-about`




## KANBAN

**任务规范**：记录成员名和任务内容
